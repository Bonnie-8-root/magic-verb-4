**English Tense Magic** 是一款专为小升初学生设计的 AI 赋能英语时态练习应用。它摆脱了传统的机械化刷题，通过“奇幻语境”和“动态信号词”帮助学生真正掌握时态背后的逻辑。

---

## 🌟 核心亮点 (Core Features)

- **AI 驱动的动态语境 (AI-Driven Context):** 利用 Google Gemini API，拒绝死板的 `every day`，生成如 `Look! The alien...` 等丰富生活语境。
- **四维时态对比 (4-Tense Comparison):** 针对大陆小升初年级教研标准，一次性对比：
  - 一般现在时 (Habit)
  - 现在进行时 (Now)
  - 一般过去时 (Past)
  - 一般将来时 (Future)
- **ADHD 友好交互 (ADHD-Friendly UX):** 采用小步分解设计、即时反馈机制和色彩心理学分级（如蓝色代表进行中，橙色代表过去）。
- **教研级纠错 (Pedagogical Accuracy):** 严格内置动词变位黑名单，规避 `singed` 等常见 AI 幻觉错误，确保学术严谨性。

## 🛠️ 技术栈 (Tech Stack)

- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
- **AI Backend:** Google AI Studio (Gemini Pro)
- **Deployment:** Vercel / GitHub Pages
- **Design Methodology:** CLIL (Content and Language Integrated Learning)

## 📖 教研逻辑说明 (Pedagogical Logic)

本项目在设计过程中参考了 CEFR A1/A2 标准及中国大陆人教版小学教材逻辑：
1. **信号词前置:** 强调“先看时间状语，再定动词形式”的解题思路。
2. **格式规范:** 强制执行句首大写、逗号后小写等书写规范，培养专业语感。
3. **主语多样性:** 随机切换主语（第三人称单数/复数），增强学生对主谓一致性的敏感度。
